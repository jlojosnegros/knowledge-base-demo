    Extra features¶

    It supports all the basic Markdown features and some additional ones like:

    -   **#Tags**
-   $$**LaTeX** math$$
-   [^**Footnotes**]
-   **[[Internal links]]**
-   **![[Filename]]** to embed notes and other files
